- ### Custom Windows Popup

  - Manufacturer Data
    - 16-bit Company Identifier ➔ ```0006```<br>
    - Raw-Data ➔ ```030008[your custom text in HEX]```<br><br>
Example Raw-Data: ```03000848656C6C6F20576F726C64```<br>
